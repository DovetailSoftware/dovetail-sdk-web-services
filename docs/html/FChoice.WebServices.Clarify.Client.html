Remarks

In addition to the classes and API documented here, there are web service client classes for each of the fcSDK API Toolkits as follows:

  • FChoice.WebServices.Clarify.Contracts.ContractsToolkitWS
  • FChoice.WebServices.Clarify.DepotRepair.DepotRepairToolkitWS
  • FChoice.WebServices.Clarify.FieldOps.FieldOpsToolkitWS
  • FChoice.WebServices.Clarify.Interfaces.InterfacesToolkitWS
  • FChoice.WebServices.Clarify.Logistics.LogisticsToolkitWS
  • FChoice.WebServices.Clarify.Quality.QualityToolkitWS
  • FChoice.WebServices.Clarify.Sales.SalesToolkitWS
  • FChoice.WebServices.Clarify.Support.SupportToolkitWS

 

For detailed information on using these toolkit client classes, please refer to the Developer Walkthrough - fcSDK Web Services.
